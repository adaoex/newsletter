
Modal windows.

Add JHtml::_('behavior.modal'); first.

Some ways to create modals:
1. Twitter's modal. As a toolbar control
	$bar = JToolBar::getInstance();
	$bar->appendButton('Popup', 'default', {title}, {url}, {width}, {height}, ... );
	See ...libraries\cms\toolbar\button\popup.php

2. Twitter's modal.
	echo JHtml::_('bootstrap.renderModal', {id}, {params}, {footer})
	See ...libraries\cms\html\bootstrap.php

3. Twitter's modal. Inline HTML markup (see TB doc)

4. SqueezeBox.
	SqueezeBox.fromElement(this, {handler:\'iframe\', size: {x: 600, y: 450}, url:\''.JRoute::_('index.php?option=com_menus&view=menutypes&tmpl=component&recordId='.$recordId).'\'})
    See ...libraries/cms/form/fields/media
